## 浏览器矩阵与证据

- 桌面：1366x768，包含 Canvas 像素检查、回退、编辑、修改导出、存档导入导出和传记下载。
- 移动：390x844，包含标签键盘导航、更多菜单、固定底栏、Canvas 与无横向溢出。
- 视口矩阵：320x568、375x667、390x844、768x1024、1024x768、1366x768、1440x900。
- 证据根目录：`test-results/iterations/v0.1/`。最新通过结果包含桌面与移动 MP4、截图、trace 和 `reports/browser-results.json`。

## 修复记录与剩余风险

- 修复了移动端旧选择器隐藏更多菜单图标的问题；该入口现有浏览器回归。
- 修复了编辑器关闭后未显式回到触发按钮的焦点行为；键盘回归已覆盖。
- 不将大体积证据提交到源码；它们是本地验收产物。运行环境必须提供本机 Chrome 和 `/opt/homebrew/bin/ffmpeg` 才能重新生成 MP4；缺失时 Playwright 仍可执行断言，但录制步骤需要恢复该工具。
