# 剧情线奖励/惩罚缺失诊断

> 诊断日期：2026-07-15
> 问题性质：剧情选项描述战斗/事件结果，但没有反馈到角色状态（等级、称号、声望、属性等）

---

## 一、诊断方法

遍历全部 275 个轮盘池、2,468 个选项，对照 `applyCommon()` 和各 handler 的处理能力，标记三种类型的选项：

- **有反馈**：选项文本含 `等级+-数字`、`【称号】`、`领域`、`魂骨`、`容貌+-数字` 等可被解析的模式
- **二择节点**：选项仅描述路线选择（"是/否"、势力选择等），没有直接奖励，但会路由到下级池
- **无反馈**（BUG）：选项描述了明确的事件结果（战胜、战败、获得物品、提升能力等），但没有任何可被解析的数值变化

## 三、详细分析

### BUG-04: 战斗结果无奖惩

**位置**：主线剧情池（嘉陵关大战、神战1/2）

**典型选项**（以嘉陵关战斗为例）：
```
"无伤，你协同天斗军团，将对面魂圣军团全部斩杀（要求80+级限定，否则重抽）"
"你一个人将对面上百魂圣军团全部斩杀（要求90+级限定，否则重抽）"
"无伤，你配合史莱克七怪摧毁对面魂圣军团"
"轻伤，无功无过（90+级可以重抽）"
"你被魂圣军团围攻，重伤战死（90+级可以重抽）"
```

**问题**：
- "无伤" vs "轻伤" vs "重伤战死" — **无区别**，全部为纯文本
- "你一个人斩杀上百" vs "配合队友" vs "无功无过" — **无区别**
- 12个嘉陵关战斗子池（剧情10.1~10.4、12.1~12.4、18.1~18.4）共 ~130 个选项，全是纯描述
- 神战战斗（剧情14/16/22/24）共 ~50 个选项，也是这样

**应该发生的**：
- 战胜 → 等级+1/+2、获得【称号】
- 战死 → 触发 lethal（部分已有"战死"字样的会被 `resultTone` 标记为 bad）
- "无伤" → 获得【轻伤无视】称号
- "轻伤" → 减等级暂时，或无事发生
- "重伤" → 等级-1 负面效果

**当前状态**：
- `lethal()` 函数能捕获"重伤战死"、"被围攻至死"等致命文本 → 游戏结束
- 但所有其他结果（战胜、秒杀、轻伤）**与"无功无过"完全等价**
- 这意味着玩家在主线的战斗抽选中，无论抽到什么战斗结果，除了死亡外**没有任何角色变化**

### BUG-05: 势力专属剧情无反馈

**位置**：`src/domain/canonAdditions.ts` 硬编码的势力剧情

**典型选项**（以佣兵团为例）：
```
【启蒙期·6-11岁】你在训练场帮新人熟悉基本阵形，被团长点名表扬
【男性路线·启蒙期·6-11岁】你和男队员组队完成第一个委托，学会分工
【女性路线·启蒙期·6-11岁】你和女队员配合护送商队，被管事多付了报酬

【成长期·12-17岁】你带队完成白银级委托，获得佣兵酒馆认可
【担当期·18岁+】你担任副团长，分配任务并审核委托报价
【精英职责·50+级】你接下暗杀任务，独闯险地并带回目标
【领队职责·70+级】你接任团长，重整章程并建立外联渠道
```

**问题**：
- 上述 ALL 选项都没有任何数值变化
- "被团长点名表扬" — 没有声望或任何效果
- "学会分工" — 没有获得技能或称号
- "被管事多付了报酬" — 没有金币或资源
- "获得佣兵酒馆认可" — 没有称号
- "接任团长" — 没有势力头衔变化

**应该发生的**：
- 完成委托 → 等级+1/+2
- 获得认可 → 获得【佣兵酒馆常客】称号
- 接任团长 → 势力头衔升级（在 `addAffiliation` 中添加）

**当前状态**：
- 势力剧情的 handler 是 `story`，结果文本走 `applyCommon()` 全局解析
- 但势力剧情选项**故意不包含**任何可被解析的模式（如 `【】`、`等级+` 等）
- 这导致 8 个势力×每个15选项 = 120个文本**全成了纯叙事，无任何游戏反馈**

### BUG-06: 学院经历无反馈

**典型选项**：
```
入学剧情:
  "顺利入学，无事发生"
  "提前与三舞相遇，留下个好印象"
  "遇到戴沐白，不打不相识"
  "遇到从窑子出来的胖子，与其打了一架"

在校2年:
  "默默无闻，一直闭关"
  "名声在外，校内小明星"
  "拜院长为师"
  "在校2年你被孤立" / "传奇海王" / "传奇快餐恋爱男/女"

比赛后去向:
  "去天斗皇家学院任职" / "留在史莱克" / "去四大元素学院任职"
  "武魂殿招募（默认叛出史莱克八怪）"
```

**问题**：
- "默默无闻" vs "名声在外" vs "拜院长为师" — 全无区别
- "留下个好印象" vs "不打不相识" — 没有任何关系数值或称号
- "拜院长为师" — 没有任何师徒关系记录
- "去天斗皇家学院任职" — 没有任何势力变化

**应该发生的**：
- "拜院长为师" → 势力/身份变化
- "名声在外" → 获得【校内明星】称号
- "被孤立" → 获得【孤独者】称号
- 比赛后去向 → 更换势力（类似 `addAffiliation`）

**当前状态**：
- "武魂殿招募（默认叛出史莱克八怪）" 的"叛出"无实际效果
- 所有入学/在校/去向选择结果没有写入任何 `context` 字段
- 角色选择了一个势力后，在校2年的所有经历对这个势力毫无影响

## 五、修复建议

### 针对 BUG-04（战斗结果）

在 `applyCommon()` 或 `case 'story':` 中增加战斗结果解析：

```
"无伤" → 等级+1
"斩杀" "秒杀" → 等级+2 或 获得称号
"轻伤" → 无变化
"重伤" → 等级-1
"战死" "战败身死" → lethal（已有，部分不匹配需补充）
```

### 针对 BUG-05（势力剧情）

在势力剧情选项文本中嵌入可被 `applyCommon()` 解析的模式：

```
"被团长点名表扬" → 改为 "被团长点名表扬，获得【佣兵新秀】称号"
"获得佣兵酒馆认可" → 改为 "获得佣兵酒馆认可，等级+1，获得【酒馆常客】称号"
"学会分工" → 改为 "学会分工配合，等级+1"
```

### 针对 BUG-06（学院经历）

同样在选项文本嵌入可解析模式：

```
"拜院长为师" → 改为 "拜院长为师，获得【院长亲传】称号"
"名声在外，校内小明星" → 改为 "名声在外，等级+1，获得【校内明星】称号"
"默默无闻，一直闭关" → 改为 "默默无闻闭关2年，等级+3"
```
