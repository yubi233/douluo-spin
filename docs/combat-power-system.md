## 一、战力值计算公式

```
战力 = 基础值 × 系数

基础值 = 等级² / 20
       + Σ 魂环战力(每个魂环)
       + Σ 武魂阶位战力(每个武魂)
       + 领域数 × 15
       + 魂骨数 × 12

系数   = 1 + 天赋系数 + 称号系数
```

## 二、各项明细

### 2.1 魂环战力

每个魂环按其年限独立计算，年限越高贡献越大：

| 年限范围 | 战力范围 | 公式 |
|----------|----------|------|
| 10年 (10-99) | 5-8 | `5 + (years-10)/90 × 3` |
| 百年 (100-999) | 9-11 | `9 + (years-100)/900 × 2` |
| 千年 (1k-9.9k) | 12-15 | `12 + (years-1000)/9000 × 3` |
| 万年 (10k-99k) | 16-20 | `16 + (years-10000)/90000 × 4` |
| 十万年 (100k-999k) | 21-30 | `21 + (years-100000)/900000 × 9` |
| 百万年 (1M+) | 31-40 | `31 + min(9, (years-1M)/1M × 9)` |

示例：
- 9个 5万年魂环：9 × ~18.2 = 164
- 9个 10万年魂环：9 × 21 = 189
- 1个百万年 + 8个十万年：35 + 8 × 21 = 203

### 2.2 武魂阶位战力

每个武魂按其六阶品级独立计算，**双生武魂加两个，不再给固定加成**：

| 阶位 | 名称 | 战力 |
|------|------|------|
| 1 | 废武魂 | +0 |
| 2 | 一般武魂 | +3 |
| 3 | 优秀武魂 | +8 |
| 4 | 顶级武魂 | +15 |
| 5 | 极品武魂 | +25 |
| 6 | 神级武魂 | +45 |

示例：
- 草鞋(废) + 双枪(顶级)：0 + 15 = 15
- 蓝电霸王龙(顶级)：15
- 极致武魂·金眼黑龙(神级)：45
- 双生 昊天锤(顶级) + 七杀剑(极品)：15 + 25 = 40

### 2.3 领域 / 魂骨（累加）

| 类型 | 每项战力 |
|------|----------|
| 每个领域 | +15 |
| 每个魂骨 | +12 |

### 2.4 天赋系数

每个天赋提供 0.5%，上限 5%（10个封顶）。

```
天赋系数 = min(天赋数量, 10) × 0.005  → 0-5%
```

### 2.5 称号系数

每个战斗相关称号（含杀/战/斗/力/破/斩/暴/狂/怒/王）提供 0.5%，上限 5%（10个封顶）。

```
称号系数 = min(战斗称号数, 10) × 0.005  → 0-5%
```

## 三、不再加成的项

| 旧项 | 状态 |
|------|------|
| 极致武魂类型固定加成 +30 | ❌ 移除 |
| 概念/变异/本体武魂类型固定加成 | ❌ 移除 |
| 双生/三生武魂固定加成 +50/+100 | ❌ 移除 |

| 配置 | 计算 | 战力 |

## 五、权重修正系数（战斗剧情）

公式未变：

```
胜利选项: multiplier = 1 + min(power / 100, 5)    上限 6x
失败选项: multiplier = max(0.05, 1 - min(power / 120, 0.95))  下限 0.05x
中性选项: multiplier = 1
```

枪械武魂越级击杀仍叠加 ×2。
